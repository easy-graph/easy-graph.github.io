Community

EasyGraph implemented community detection methods for use.

Modularity

modularity(G, communities[, weight])

Returns the modularity of the given partition of the graph.

Modularity_Max_Detection

greedy_modularity_communities(G[, weight])

Communities detection via greedy modularity method.

Community Detection

LPA(G)

Detect community by label propagation algorithm Return the detected communities.

SLPA(G, T, r)

Detect Overlapping Communities by Speaker-listener Label Propagation Algorithm Return the detected Overlapping communities.

HANP(G, m, delta[, threshod, hier_open, ...])

Detect community by Hop attenuation & node preference algorithm

BMLPA(G, p)

Detect community by Balanced Multi-Label Propagation algorithm

Motif Detection

enumerate_subgraph(G, k)

Returns the motifs.

random_enumerate_subgraph(G, k, cut_prob)

Returns the motifs.
