easygraph.MultiDiGraph.all_neighbors

MultiDiGraph.all_neighbors(node)

Returns an iterator of a nodes neighbors, including both successors and predecessors.

Parameters

node (object) The target node.

Returns

neighbors An iterator of a nodes neighbors, including both successors and predecessors.

Return type

iterator

Examples

>>> G = eg.Graph()
>>> G.add_edges([(1,2), (2,3), (2,4)])
>>> for neighbor in G.all_neighbors(node=2):
...     print(neighbor)
