class easygraph.datasets.hypergraph.walmart_trips.walmart_trips(data_root=None, local_path=None)[source]#

Bases: object

A class for loading and processing the Walmart trips hypergraph dataset.

This class fetches hyperedge, node label, and label name data from predefined URLs, processes the data, and generates a hypergraph representation. It also provides access to various dataset attributes through properties and indexing.

data_root#

The root URL for the data. If provided during initialization, it is set to “https://”; otherwise, it is None.
